# Partnership Proposal - Albedo's Return

A professional proposal website for **Ann Tran Perfect** of **Albedo's Return**.

## Overview

This is a static HTML/CSS proposal site designed for presenting two partnership options:

1. **Fractional Growth Partner** - Foundation + Monthly Strategy
2. **Growth Participation Partner** - Lower Retainer + Commission

## Features

- Clean, professional design using Poppy Marketing brand colors
- Side-by-side option comparison
- One-click email acceptance buttons
- Mobile responsive
- Print-friendly styles
- No JavaScript dependencies (pure HTML/CSS)

## Deployment

### GitHub Pages

1. Create a new repository on GitHub
2. Push this folder to the repository
3. Enable GitHub Pages in repository settings
4. Choose `main` branch as source

### Vercel / Netlify

Simply connect your Git repository and deploy - no configuration needed.

Clicking "Accept Option X" generates a pre-filled email to:
- **To:** email@poppy-marketing.com

- **Body:** Pre-filled acceptance message
